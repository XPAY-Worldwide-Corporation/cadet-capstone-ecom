
Customer
GET /customers: Get all customers.
GET /customers/
: Get a specific customer by ID.
POST /customers: Create a new customer.
PATCH /customers/
: Partially update an existing customer by ID.
DELETE /customers/
: Delete a customer by ID.
PUT /auth/customers/forgot-password: Initiate the password reset process for a customer.
POST /auth/customers/reset-password: Reset a customer's password using a token.
PATCH /auth/customers/update-password: Update a customer's password while logged in.
