
Users
GET /users: Get all users
GET /users/
: Get a specific user by ID
POST /users: Create a new user
PATCH /users/
: Update an existing user by ID
DELETE /users/
: Delete a user by ID
PUT /auth/forgot-password: Initiate the password reset process (public route)
POST /auth/reset-password: Reset the user's password using a token (public route)
PATCH /auth/update-password: Update the authenticated user's password (protected route)
